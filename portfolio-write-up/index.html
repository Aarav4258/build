Overview

Function & Purpose

My program allows the user to guess a word from a random list of words. Each time the user guesses, they get feedback on whether any of the letters they guessed was correct. Initially, all the user knows is how many letters are in the word, but each guess gives them more information.

Design Process

I had to break my design into steps, including:

  • Representing the list of words the user guessed as a list in TypeScript
  • Selecting a random word from the list for the user to guess.
  • Getting user input from an HTML input.
  • Comparing the user input to the word to show which letters were correct.
  • Allowing the user to start a new game when finished by choosing a new random word.

What does this code do?

This section of my code (1) defines a list and (2) defines a procedure which returns a random word from the list.

In my game, that procedure is called once at the start of the game and then again each time the user starts a new game.

Explanation

Defining a List: To define a constant in TypeScript I use the const keyword, so the first line here defines a new constant named words.

Each item in words is a string, which is defined using quotation marks.

Defining a procedure The getRandomAnswer procedure takes no arguments and returns a random word from the list each time.

First, I get a random index for the list. In JavaScript, Math.random() returns a random number between 0 and 1. By multiplying this number by the length of the list and then taking the "floor" of that number, I get an integer between 0 and the length of the list.

Then I select the list item at index, so e.g. if the list has 10 items and I get a 4 for my randomIndex, then I would return the 5th item of the list from getRandomAnswer() using the syntax for selecting an item from a list: words[randomIndex].
